# Simple Stock Viewer

## .NET Core Angular Project

### Angular Project

Open Terminal/Command Prompt
- > cd \SimpleStockClient 
- > npm install
- > ng serve --open


### .Net Core WebAPI

- Open \SimpleStockServer folder
- Open SimpleStockServer.sln using Visual Studio 2019 or latest
- Wait for the package updates
- Run the Project
